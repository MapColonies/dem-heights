# dem-heights

This is a repo for extracting elevation info in a given location/position.

## Run Locally

Clone the project:

```bash

git clone https://link-to-project

```

Go to the project directory:

```bash

cd my-project

```

Install dependencies:

```bash

npm install

```
Install git hooks:

```bash

npx husky install

```

Start the server:

```bash

npm start

```
